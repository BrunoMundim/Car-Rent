Descrição

Car Rent é uma API para controle de aluguel de veículos, ela permite listar veículos, para que usuários façam suas contas e aluguem carros.

Funcionalidades

  • Cadastro usuários
  • Cadastro de veículos e listagem por: Marca, Modelo e listar todos.
  • Aluguel de veículos e devoluções, mantendo no banco de dados o estado atual dos veículos.
  • Envio de emails confirmando aluguel e devolução.

Tecnologias utilizadas

  • Java
  • Spring Boot
  • Spring Data JPA
  • Spring Security
  • PostgreSQL
  • JUnit
